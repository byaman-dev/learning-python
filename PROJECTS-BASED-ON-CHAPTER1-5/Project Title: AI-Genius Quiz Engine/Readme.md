# 🤖 AI-Genius: Intelligence & Security Assessment Engine (v2.0)

**AI-Genius** is a professional-grade Command Line Interface (CLI) application designed to evaluate domain knowledge in **Artificial Intelligence** and **Cybersecurity**. This project is the culmination of Chapters 1-5 of the Python Masterclass, moving from simple scripts to a scalable, data-driven architecture.

---

## 🚀 Key Features in Version 2.0
* **Bi-Topic Intelligence:** Switch between specialized modules for Cybersecurity and AI Fundamentals.
* **Exhaustive Knowledge Base:** 20 high-quality questions (10 per category) designed for technical accuracy.
* **Dynamic Module Injection:** Uses a "Menu-to-Data" mapping logic that allows new topics to be added without modifying the core engine code.
* **Input Sanitization Pipeline:** Processes all user responses through `.strip().upper()` to ensure a frictionless, case-insensitive experience.
* **Real-time Analytics:** Provides final score tallying and success-rate percentage calculation.

---

## 🏗️ Technical Architecture (Mastery 1-5)
This project demonstrates advanced proficiency in the following Python fundamentals:

### 1. Nested Data Structures (Ch 5)
The heart of the app is a **"Dictionary of Dictionaries,"** showcasing complex data retrieval and organization. This allows the program to store questions, options, and answers in a structured hierarchy.

### 2. List Iteration (Ch 4)
Efficiently manages multiple-choice options using list-based displays within the dictionary structure.

### 3. Immutability & Normalization (Ch 3)
Handles user input through string transformation methods to prevent logic errors and ensure "crash-proof" execution.

### 4. Modular Control Flow (Ch 1 & 2)
Organizes code into reusable blocks and manages application "state" (scoring) through integer and float variables.
