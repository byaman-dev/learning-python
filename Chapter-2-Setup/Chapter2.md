# 🛠️ Chapter 2: Setting Up the Local Environment
> **"A clean environment is the secret to clean code."**

In this chapter, I moved from browser-based coding to setting up a professional developer environment on my local machine. This is where the real journey begins.

---

## 📥 1. Python Installation
I installed Python 3.12+ from [python.org](https://www.python.org).
* **Crucial Step:** I checked the box **"Add Python to PATH"** during installation. This allows me to run Python commands directly from the terminal/command prompt.

## 💻 2. VS Code Configuration
I am using **Visual Studio Code (VS Code)** as my primary editor.
* **Why VS Code?** It's lightweight, powerful, and has a built-in terminal.
* **Extension:** Installed the official **Python extension** by Microsoft for features like IntelliSense (code completion) and linting.

---

## 🧪 3. Exploring the REPL
I practiced using the **REPL** (Read-Eval-Print Loop).
* **How to enter:** Typed `python` in the terminal to get the `>>>` prompt.
* **What I learned:** The REPL is an interactive shell that executes code instantly. It is perfect for testing quick math, logic, or trying out a new function without saving a `.py` file.
* **How to exit:** Typed `exit()` or used `Ctrl + Z`.
